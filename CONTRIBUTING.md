# Contribuindo para Estudo Site Filmes

Obrigado por seu interesse em contribuir para o Estudo Site Filmes!

## Como Contribuir

1. Faça um fork do repositório.
2. Crie um branch para sua feature ou correção de bug (`git checkout -b feature/nova-feature`).
3. Commit suas alterações (`git commit -m 'Adiciona nova feature'`).
4. Faça o push para o branch (`git push origin feature/nova-feature`).
5. Abra um Pull Request.

## Requisitos de Contribuição

- Siga o estilo de código definido pelo ESLint.
- Adicione testes para suas alterações sempre que possível.
- Certifique-se de que todos os testes estão passando antes de enviar seu PR.

## Código de Conduta

Este projeto adota o [Código de Conduta do Pacto do Contribuinte](https://www.contributor-covenant.org/version/2/0/code_of_conduct/). Ao participar, você está de acordo em manter um ambiente acolhedor e respeitoso para todos.

## Contato

Para qualquer dúvida, abra uma issue ou entre em contato diretamente pelo GitHub.
